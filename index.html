Omi
开放现代的Web组件化框架
Github

  • 局部CSS

    HTML+ Scoped CSS + JS组成可复用的组件。不用担心组件的CSS会污染组件外的,Omi会帮你处理好一切

  • 超小的尺寸

    7 kb (gzip)
    虽然文件尺寸小，但是兼容性很好，兼容IE8及IE8以上版本，动画性能所有框架最快

  • 更自由的更新

    每个组件都有update和updateSelf方法，自由选择你认为最佳的更新时机和更新方式

  • 模板引擎可替换

    开发者可以重写Omi.template方法来使用任意模板引擎

  • 完全面向对象

    函数式和面向对象各有优劣，Omi使用完全的面向对象的方式来构建Web程序

  • ES6+ 和 ES5都可以

    Omi提供了ES6+和ES5的两种开发方案。你可以自有选择你喜爱的方式

  • npm install omi-cli -g
  • omi init your_project_name
  • cd your_project_name
  • npm start
  • npm run dist
